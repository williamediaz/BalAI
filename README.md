# BalotoAI

Solución Clean Architecture en .NET 10 llamada BalotoAI.

Esta carpeta contiene un scaffold inicial con los proyectos:
- BalotoAI.Domain
- BalotoAI.Application
- BalotoAI.Infrastructure
- BalotoAI.Web (Razor Pages)
- BalotoAI.Tests (xUnit)

Incluye configuración básica para EF Core (SQL Server), MediatR (CQRS), FluentValidation, Serilog, Swagger, ML.NET, y contenedores Docker.

Siguientes pasos (pendientes):
- Implementar algoritmos estadísticos completos (Bayes, Markov, Monte Carlo, Genético)
- Integrar ML.NET con modelos entrenados
- Dashboard con Chart.js y endpoints REST
- Pruebas unitarias y de integración exhaustivas
